Class UserInitialsFieldInstruction

OpenLanguage.WordprocessingML.FieldInstruction.Ast

Represents a strongly-typed USERINITIALS field instruction. Retrieves the current user's initials or, if field-argument is present, the initials specified by text in field-argument. Specifying a field-argument shall not change the initials of the current user.
