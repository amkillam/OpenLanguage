Class AlphabeticHexevigesimalProvider

Namespace
OpenLanguage.Utils
Assembly
OpenLanguage.dll

Provides formatting services to convert a UInt64 to and from an Excel-style alphabetic column string (e.g., 1 -> "A", 27 -> "AA").

Usage for formatting: UInt64 value = 27; string formatted = value.ToString("AH", new AlphabeticHexevigesimalProvider()); // "AA"

Usage for parsing: string column = "AA"; UInt64 parsed = AlphabeticHexevigesimalProvider.Parse<UInt64>(column); // 27
