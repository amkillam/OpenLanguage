OpenLanguage.WordprocessingML.FieldInstruction

Represents a strongly-typed MACROBUTTON field instruction. Allows the macro or command designated by text in field-argument-1 to be run. Text in field-argument-2 designates the text or graphic to appear as the "button" that is selected to run the macro or command.
