Class HyperlinkFieldInstruction

Namespace
OpenLanguage.WordprocessingML.FieldInstruction
Assembly
OpenLanguage.dll

Represents a strongly-typed HYPERLINK field instruction. When selected, causes control to jump to the location specified by text in field-argument. That location can be a bookmark or a URL.
